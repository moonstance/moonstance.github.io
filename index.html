DbStudio

DbStudio is a lightweight, developer-friendly C# scripting environment for interacting with RavenDB. Write and run powerful LINQ queries or modify data directly against your RavenDB using live C# code, powered by Roslyn and using the outstanding Roslynpad. It's fast, intuitive, and perfect for developers who want the freedom of scripting with the comfort of .NET.
